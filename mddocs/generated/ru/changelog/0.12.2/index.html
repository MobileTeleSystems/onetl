0.12.2 (2024-11-12)¶

Улучшения¶

  • Изменен Spark jobDescription для DBReader и FileDFReader с DBReader.run() -> Connection на Connection -> DBReader.run().

Исправления ошибок¶

  • Исправлен результат log_hwm для KeyValueIntHWM (используется Kafka). (#316)
  • Исправлено сокрытие значений Kafka.addresses в логах с уровнем INFO в log_collection. (#316)

Зависимости¶

  • Разрешено использование etl-entities==2.4.0.

Изменения только в документации¶

  • Исправлены ссылки на документацию по типам даты и времени MSSQL.

Базовый пакет `onetl` содержит:

* `DBReader`, `DBWriter` и связанные классы
* `FileDownloader`, `FileUploader`, `FileMover` и связанные классы, такие как файловые фильтры и лимиты
* `FileDFReader`, `FileDFWriter` и связанные классы, такие как форматы файлов
* Стратегии чтения и классы HWM
* Поддержка плагинов

Он может быть установлен через:

```bash
pip install onetl
```

!!! warning

    Этот метод НЕ включает в себя никаких подключений.

    Этот метод рекомендуется для использования в сторонних библиотеках, которым требуется установка `onetl`, но не использующих ее классы подключений.

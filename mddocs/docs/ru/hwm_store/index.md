# HWM

Начиная с onETL v0.10.0, классы `HWMStore` и `HWM` были перемещены в отдельную библиотеку {{ etl_entities_link }}.

Единственным классом, который остался неизменным, является [`yaml-hwm-store`][yaml-hwm-store], **который используется по умолчанию** в onETL.

Другая известная реализация - [HorizonHWMStore](https://horizon-hwm-store.readthedocs.io/).

* [YAML HWM Store][yaml-hwm-store]

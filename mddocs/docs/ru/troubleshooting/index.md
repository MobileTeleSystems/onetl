# Решение проблем { #troubleshooting-main }

В случае ошибки, пожалуйста, следуйте инструкциям ниже:

- Прочитайте логи или сообщения об ошибках, с которыми вы столкнулись.
  - Если логи Python недостаточно подробны, [увеличьте уровень логирования][logging].
  - Если логи Spark недостаточно подробны, [увеличьте уровень логирования][troubleshooting-spark].
- Прочитайте документацию, относящуюся к классу или методу, который вы используете.
- [Погуглите](https://google.com) сообщение об ошибке и внимательно прочитайте результаты поиска:
  - Ответы на [StackOverflow](https://stackoverflow.com/).
  - Документацию [Spark](https://spark.apache.org/docs/latest/).
  - Документацию базы данных или файловой системы, к которой вы подключаетесь.
  - Документацию используемого коннектора.
- Поищите известные [проблемы](https://github.com/MobileTeleSystems/onetl/issues) или создайте новую.
- Всегда используйте самые последние версии onETL, PySpark и пакетов коннекторов, [совместимые с вашей средой][install-spark].

- [spark][troubleshooting-spark]
